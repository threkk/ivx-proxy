FAQ

How does this work?

iVoox Originals podcasts can not be listened without using their application. The proxy will use public information to build an alternative feed that can be used in your favourite application instead of theirs. This is, freedom for you to choose your application instead of being forced to use theirs.

What podcast can I submit? The podcast MUST
  • Be hosted in iVoox
  • Be an iVoox original
  • Have a public RSS feed
Got the link, now what? You can remove the old feed from the podcast application and use the new one instead. In order to add/remove the podcast, check the instructions of your application.
Is this legal? This is within some grey area. The information used is publicly available information in the website. In order to build the alternative feed, iVx-Proxy does not make use of any credential or restricted access to the original feed. It falls within the same legislation as web scraping. You can achieve the same result with the developer tools and patience, although you never know...
